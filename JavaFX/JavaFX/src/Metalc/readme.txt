El siguiene proyecto requiere la Libreria JavaFX, ademas es una implementacion de INTERFAZ, la mayoria de los botones interactivos
no estan codificados.
Tambien tener en cuenta que para el inicio de sesion Si esta codificado y se requiere de una base de datos ya sea local o en red
la cual debe tener una tabla usuarios(ver logica del codigo y sentencias SQL).
Usa libreria de Conector JBD para conectar a base de datos LOCAL de Mysql.
El modulo de ENVIOS No esta nada TRABAJDO, asi que no ejecutarlo o terminarlo.
